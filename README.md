# toy-process-manager

simple process manager GUI implemented with electron.

```sh
# development
pnpm i
npm run dev

# package and run
npm run build
npm run package
./build/toy-process-manager-0.0.0.AppImage

# release (this command just pushes tag and then CI will publish new version)
npm run release
```
